# DoggoReminderUI

## Descrição

Este é o front-end da aplicação que consome a API em Python Flask para gerenciar registros de aumigos, rotinas de alimentação e notificações de alimentação. Este front-end oferece uma interface amigável para os usuários interagirem com a API.

## Funcionalidades

- Cadastro de usuário.
- Login de usuário.
- Visualizar a lista de aumigos registrados.
- Adicionar um novo aumigo.
- Visualizar detalhes de um aumigo específico.
- Atualizar informações de um aumigo.
- Remover um aumigo.
- Visualizar a lista de rotinas de alimentação registradas.
- Adicionar uma nova rotina de alimentação.
- Visualizar detalhes de uma rotina de alimentação específica.
- Criar uma nova notificação de alimentação para um cachorro em um horário específico.
- Receber notificações no dispositivo de acordo com as rotinas criadas.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Biblioteca/Bibliotecas de requisições HTTP (Fetch)

## Como Executar

1. Clone este repositório para o seu ambiente local:

    ```
    git clone https://github.com/Rodrigocambraia14/DoggoReminder.UI.git
    ```

2. Navegue até o diretório do projeto:

    ```
    cd DoggoReminderUI
    ```

3. Abra o arquivo HTML principal em um navegador da web para iniciar a aplicação.

## Observações

1. Este projeto é apenas um MVP, portanto, não possui recursos como autenticação JWT, OAuth2, responsividade mobile, endpoints de funcionalidades complexas e afins.
